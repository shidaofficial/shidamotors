# shidamotors
Shida Motors, lüks ve ikinci el araçların sergilendiği modern bir otomobil galerisi sitesidir. Kullanıcılar marka ve fiyat aralığına göre araç filtreleyebilir, araç detaylarını inceleyebilir ve kendi ilanlarını yayınlayabilir. Firebase altyapısıyla giriş ve kayıt sistemi de mevcuttur. Site HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.
